# Music-Mobile-App

App nghe nhạc online đơn giản cho android. <br>
Server host: https://vn.000webhost.com/members/website/list, PhpMyAdmin

## Features
* Load dữ liệu trực tuyến
* Phát nhạc trực tuyến 
* Sử dụng đơn giản
* Tôc độ ổn định

## Thư viện
* Retrofit
* Picasso API
* Một số thư viện design
